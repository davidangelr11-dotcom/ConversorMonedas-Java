# 💱 Conversor de Monedas en Java

Proyecto desarrollado como práctica de aprendizaje en Java usando IntelliJ IDEA.

## 📌 Descripción

Este programa permite convertir valores entre diferentes monedas utilizando una API de tasas de cambio en tiempo real.

El usuario puede seleccionar:

* Moneda de origen
* Moneda de destino
* Cantidad a convertir

El sistema consulta una API externa y muestra el resultado en consola.

---

## 🚀 Tecnologías utilizadas

* Java 17
* IntelliJ IDEA
* Maven
* Gson (para manejo de JSON)
* API ExchangeRate

---

## 🧠 Conceptos aprendidos

Durante el desarrollo practiqué:

* Programación orientada a objetos (POO)
* Consumo de APIs REST
* Manejo de JSON
* Uso de Maven
* Control de versiones con Git y GitHub

---

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Abrir en IntelliJ IDEA
3. Ejecutar la clase:

```
Principal.java
```
